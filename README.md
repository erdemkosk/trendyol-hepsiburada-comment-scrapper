## Prerequisites

- node 14.15.0
- npm 6.14.8

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Docker

```sh
docker build . -t <your username>/express-boilerplate
docker run --env-file .env -p 5000:5000 <your username>/express-boilerplate 
```
